Mimetic In A Nutshell

Mimetic.scale
The layout is constructied to a fixed width or height, the layout, fonts and images are scaled to fit any device. Your website is of full width/ height on eveery device.
Mimetic.baseLayer (.base)
This is the most simplified layer of the website without content. This layer only contains layout tags and maybe color fills, it is not appriximate.
Mimetic.interfaceLayer (.ui)
The interface layer defines UI elements primarilay using SVG.
Mimetic.contentLayer (.con)
This layer is strictly for content, the "layout" shoud only consist of the main content
